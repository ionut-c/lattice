Checkbox

LATTICE offers styling for checkbox inputs

data-interface = "lattice"
accepts lattice in order to call the lattice interface
data-behaviour = toggle
accepts toggle in order to call the checkbox
data-size = "1"
accepts numerical values
data-label = empty
if sets creates a clickable label for the input
<input type="checkbox" data-interface="lattice" data-behaviour="checkbox" data-size="1" data-label="Label Here">

Toggle

LATTICE offers styling for toggle switch

data-interface = "lattice"
accepts lattice in order to call the lattice interface
data-behaviour = toggle
accepts toggle in order to call the toggle switch
data-size = "1"
accepts numerical values
data-checked = Yes
accepts alphanumeric characters for the checked input
data-unchecked = No
accepts alphanumeric characters for the unchecked input
<input type="checkbox" data-interface="lattice" data-behaviour="toggle" data-checked="Yes" data-unchecked="No" data-size="1">

File Upload

SUIT offers styling for file upload inputs

data-interface = "lattice"
accepts suit in order to call the suit interface
data-label = empty
if sets creates a clickable label for the input
accept = audio/* | video/* | image/*
if the accept attr is set it creates an icon illustrating file type (audio/*, video/*, image/*)
<input type="file" data-interface="lattice" data-label="Label Here">
