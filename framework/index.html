S.U.I.T. Shared User Interface Tools

Checkbox

SUIT offers styling for checkbox inputs

data-interface = "suit"
accepts suit in order to call the suit interface
data-size = "1"
accepts numerical values
data-label = empty
if sets creates a clickable label for the input
<input type="checkbox" data-interface="suit" data-size="1" data-label="Label Here">
