UI tools

This website is an open source collection of UI tools. Which are still being developed.

SASS Plate

SASS plate is a variation of HTML5 Boilerplate to help you get started on a sass based project. You will need compass installed to be able to use SASS.

SUIT Framework

Get stylish Checkboxes and Radio buttons with the suit framework.

Paxy

Paxy or Parallax Y is a UI technology for animating CSS properties in response to how the user is scrolling vertically.
